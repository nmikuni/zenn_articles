# zenn_articles

[Zenn](https://zenn.dev/n_mikuni/articles/) に投稿している記事の **一部** をこのリポジトリで管理しています。

## 管理方法

[Zenn CLI で記事・本を管理する方法 / GitHub リポジトリとの連携](https://zenn.dev/zenn/articles/connect-to-github) に則り、`main` ブランチへの push で Zenn 側に自動デプロイされます。

## Claude Code 用スキル

記事の作成・編集手順は Claude Code のスキル [`.claude/skills/zenn-article/SKILL.md`](.claude/skills/zenn-article/SKILL.md) にまとめてあります。Claude Code を使うときはここを参照してください。
